# 介绍

Tabris.js是一个移动端框架，可以让你用*JavaScript*或*TypeScript*或*JSX*编写的*单一代码*来开发*iOS、Android、Windows*应用。
它提供接近*原生的组件*和功能，在每个平台上提供原生性能外观和体验。

<img src="https://user-images.githubusercontent.com/255637/28335472-c5f154aa-6bfe-11e7-905d-742218d40b19.png" alt="Bookstore example on Windows" width="180" style="margin: 5px" />
<img src="https://user-images.githubusercontent.com/255637/28335465-c37e7bc6-6bfe-11e7-92c8-97ce71ad6d90.png" alt="Bookstore example on iOS" width="180" style="margin: 5px" />

Tabris.js在设计时就考虑到了兼容性和可扩展性。
这个框架暴露了各种周知的JavaScript API，像 [Timers](https://developer.mozilla.org/en-US/docs/Web/API/WindowOrWorkerGlobalScope/setTimeout), [Console](https://developer.mozilla.org/en-US/docs/Web/API/Console), [Fetch](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API), 和 [WebSockets](https://developer.mozilla.org/en-US/docs/Web/API/WebSocket)。
你可以从[npm](http://npmjs.com/)生态系统和[Cordova](https://cordova.apache.org/plugins/)插件中获取的JavaScript库来访问不能由Tabris.js直接访问的设备功能。

## 第一步

你可以在不在电脑上安装任何东西的情况下来试试Tabris.js。

* 在你的设备上安装[Tabris.js开发者应用](developer-app.md)，并浏览应用中的示例。
* 在线上试试[Tabris.js Playground](https://tabrisjs.com/playground)上的一个简单的Tabris.js应用的JavaScript代码。(https://tabrisjs.com/playground).
* 通过扫描playground页面中的二维码来将你编辑过的版本加载到开发者应用中。

跟着[快速开始指南](getting-started.md)来开发一个真正的Tabris.js应用吧。
我们还有一本优秀的[电子书](https://tabrisjs.com/downloads/ebook/tabrisjs-2.1.0.pdf)来解释如何创建、部署和测试你的第一个Tabris.js应用。
